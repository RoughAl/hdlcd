Detailed Description

Class ClientAcceptor.

This class is responsible to accept incoming TCP connections. For each inbound TCP connection, a ClientHandler object is creating taking full responsibility of the connection. These TCP connections originate from clients that access the HDLCd and contain PDUs of the HDLCd access protocol.
